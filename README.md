# UniFi Network Controller – Docker Stack

Portable, reproducible UniFi Network Application deployment using Docker.

## Features
- MongoDB-backed UniFi Network Application
- Designed for mobile timing / comms boxes
- Optional Let's Encrypt (DNS-01 via Cloudflare)
- Backup & restore scripts
- VPS- or laptop-friendly
